## What is Hugo?

Hugo is a fast and flexible static site generator written in Go. It's perfect for blogs, portfolios, and documentation sites.

### Key Features

- **Lightning fast** - Builds sites in milliseconds
- **Flexible** - Use any content structure you want
- **Powerful** - Rich content management features
- **No dependencies** - Single binary, no databases needed

## Writing in Markdown

- **Bold text** using `**text**`
- *Italic text* using `*text*`
